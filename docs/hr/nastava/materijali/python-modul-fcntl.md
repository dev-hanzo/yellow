# Python: usluge specifične za operacijske sustave slične Unixu: pozivi fcntl() i ioctl()

- `module fcntl` ([službena dokumentacija](https://docs.python.org/3/library/fcntl.html)) nudi pristup pozivima `fcntl()` i `ioctl()`
- `fcntl.fcntl()`
- `fcntl.flock()`
- `fcntl.ioctl()`
- `fcntl.lockf()`
