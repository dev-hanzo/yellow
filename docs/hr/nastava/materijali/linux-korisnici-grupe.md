# Korisnici i grupe

!!! hint
    Za dodatne primjere naredbi proučite [stranicu Users and groups na ArchWikiju](https://wiki.archlinux.org/title/Users_and_groups).

- naredba `useradd`
- naredba `usermod`
- naredba `userdel`
- naredba `groupadd`
- naredba `groupmod`
- naredba `groupdel`

## Promjena vlasnika i grupe

- naredba `chown`
- naredba `chgrp`

## Sposobnosti datoteka

- engl. *file capabilities*
- naredba `setcap`
- naredba `getcap`
